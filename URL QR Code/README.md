## 🔗 URL QR Code — Ready 🚀

A fully featured URL / Text QR Code Generator built with HTML, CSS, and Vanilla JavaScript.
This project allows users to instantly generate QR codes for links or text with customization options.


## 🔗 Generate QR codes from URLs or text:<br>

📏 Adjustable QR code size<br>

🎨 Custom QR color selection<br>

🛡️ Select error correction level (L, M, Q, H)<br>

🌙 Dark mode toggle<br>

📱 Fully responsive design<br>

⚡ Fast generation using QR API<br>

## 🛠️ Built With

HTML5
CSS3
Vanilla JavaScript
QRServer API

## 📖 How It Works

Enter a URL or text<br>
Choose QR size<br>
Select color<br>
Pick error correction level<br>
Click Generate QR Code

<br>Your QR code will appear instantly.
